## Docusaurus Guide

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (>= 14.x)
- **npm** or **yarn**

### Installation

To set up the project, install dependencies:

```bash
cd my-docs
npm install
```

### Running the Web App

Start the development server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.
